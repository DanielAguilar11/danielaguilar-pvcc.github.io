    Operating Systems Terminology

Term Definition
Operating System The software that supports a computer's basic functions, such as scheduling tasks, executing applications, and controlling peripherals.
Job Queue A list of tasks to be carried out in turn by an operating system.
FIFO First In, First Out (FIFO) is an accounting method in which assets purchased or acquired first are disposed of first. FIFO assumes that the remaining inventory consists of items purchased last.
Interactive Processing In a computer system, the process of interacting directly with a program and/or data for immediately modifying, retrieving and/or displaying information.
Time-Sharing The operation of a computer system by several users for different operations at the same time.
Multiprogramming The running of two or more programs or sequences of instructions simultaneously by a computer with more than one central processor.
Application Software Application software, also called application program, is a software designed to handle specific tasks for users.
System Software System software is a type of computer program that is designed to run a computer's hardware and application programs.
Utility Software Utility software helps users configure, analyse, optimise and maintain a computer. This software normally consists of small programs which are thought of as part of the operating system (OS) as they often come built in with the OS.
Graphical User Interface (GUI) A visual way of interacting with a computer using items such as windows, icons, and menus, used by most modern operating systems.
Shell Shell is a UNIX term for the interactive user interface with an operating system. The shell is the layer of programming that understands and executes the commands a user enters. In some systems, the shell is called a command interpreter.
Kernel Kernel is central component of an operating system that manages operations of computer and hardware. It basically manages operations of memory and CPU time. It is core component of an operating system.
Device Driver A device driver is a special kind of software program that controls a specific hardware device attached to a computer.
File Manager A file manager or file browser is a computer program that provides a user interface to manage files and folders.
Memory Manager In operating systems, memory management is the function responsible for managing the computer's primary memory. The memory management function keeps track of the status of each memory location, either allocated or free.
Virtual Memory A virtual manager is a tool or resource that allows for the management of more than one computer from a single workstation.
Boot Strapping In Software development it refers to the development of successively more complex, faster programming environments.
ROM Read-only memory, or ROM, is a type of computer storage containing non-volatile, permanent data that, normally, can only be read, not written to.
Deadlock A deadlock is a situation in which two computer programs sharing the same resource are effectively preventing each other from accessing the resource, resulting in both programs ceasing to function.
Spooling In computer processes, spooling is the practice of holding data in temporary storage for execution by another device or program. In modern systems, spooling is often used when data needs to be transferred from a program to a peripheral (such as a printer).
Super User (or Administrator) A user of a computer system with special privileges needed to administer and maintain the system; a system administrator.
Multi-Factor Authentication Multi-factor Authentication (MFA) is an authentication method that requires the user to provide two or more verification factors to gain access to a resource such as an application, online account, or a VPN.
